# Sobre o Projeto

Este projeto foi desenvolvido como parte do desafio prático da plataforma DIO, com o objetivo de aplicar conceitos de Segurança da Informação e Engenharia Social utilizando a ferramenta SEToolkit (Social-Engineer Toolkit)

A proposta consiste em demonstrar, em ambiente controlado e para fins exclusivamente educacionais, como ataques de phishing podem ser simulados através da clonagem de páginas de login, permitindo compreender vulnerabilidades relacionadas ao comportamento humano e à segurança digital.

# Objetivos do Projeto
- Aplicar conceitos de Engenharia Social na prática;
- Utilizar o SEToolkit para criação de páginas clonadas;
- Compreender o funcionamento de ataques de phishing;
- Demonstrar técnicas de captura de credenciais em laboratório;
- Desenvolver documentação técnica clara e organizada;
- Utilizar GitHub para versionamento e apresentação do projeto.

# Ferramentas Utilizadas
- Kali Linux
- SEToolkit
- Terminal Linux
- Git & GitHub
